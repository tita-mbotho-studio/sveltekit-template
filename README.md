# SvelteKit Website Template

A reusable SvelteKit template for landing pages, showcase websites, portfolio sites, and business brochure sites.

## Structure

- `app/` - SvelteKit application
- `.github/workflows/ci.yml` - CI workflow
- `LICENSE` - restrictive copyright license

## Conventions

- PascalCase for Svelte component files
- lowercase for config and utility files
- generic section naming for reuse
- no client-specific assets in the template

## Usage

1. Create a new repository from this template
2. Update site content in `app/src/lib/config/site.ts`
3. Replace placeholder assets in `app/static/img/`
4. Run locally and build
